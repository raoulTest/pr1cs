# Fix Vite allowedHosts Error

## Problem
The error "Blocked request. This host ("app.alpha.openscaler.net") is not allowed" occurs because `allowedHosts` is only configured for `preview` mode, but the app is running in `server` (dev) mode.

## Current State
In `apps/web/vite.config.ts`:
- `preview.allowedHosts` has `"app.alpha.openscaler.net"` (line 13)
- `server` section only has `port: 3001` (lines 9-11)

## Solution
Add `allowedHosts` to the `server` configuration to match the `preview` configuration.

## File to Modify
- `apps/web/vite.config.ts`

## Change
```ts
server: {
  port: 3001,
  allowedHosts: ["app.alpha.openscaler.net"],
},
```

## Verification
1. Restart the dev server
2. Access `app.alpha.openscaler.net` - should no longer show the blocked request error
